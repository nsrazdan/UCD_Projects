# ECS 150 Project 1: Simple Shell

- This README is to introduce the organization of our project
- This is **not** the report file
- Read below to find path to report

## Project Files
- All project files are in the `/src` directory
- **To compile**: run `make` while in `/src`
- **To run**: run `/sshell` after compiling
- **To clean**: run `make clean`
- Header files in `/include` subdirectory
- Object files in `/obj` subdirectory
- C Files, makefile, and executable are in `/src` directory

## Informational Files
- All other informational files are in the home directory (.)
- Handin does not allow to be in a subdirectory
